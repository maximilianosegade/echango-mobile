# echango-FrontEnd

Requerimientos minimos:

Guía instalación Windows

Paso 0

Consola piola (por si no gusta la default, powercmd me pinchó por todos lados)
https://sourceforge.net/projects/console/files/latest/download

Ultima versión de Java
http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html

Apache ANT, bajar el zip y descomprimir en c:\Proyecto\apache-ant
http://ant.apache.org/bindownload.cgi

Última versión de la SDK de android (está bien abajo ne la página)
http://developer.android.com/sdk/index.html#downloads

Emulador de dispositivos android
https://www.genymotion.com/download/

Visual Studio, ide de desarrollo
https://code.visualstudio.com/

Node.js versión 4.x
https://nodejs.org/en/

Usuario de Ionic
https://apps.ionic.io/signup

Variables de entorno
JAVA_HOME = C:\Program Files (x86)\Java\jdk1.7.0_67
ANDROID_HOME = C:\Program Files (x86)\Android\android-sdk
PATH = C:\apache-ant\bin;%JAVA_HOME%\bin;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools

Paso 1
Abris la consola copada y tiras
npm install -g cordova
Se pone a bajar un millón de cosas, baja descomprime, copia, baja descomprime copia pega, 5 minutos o más, fácil.

Paso 2
npm install -g ionic
El primer caracter de la linea siguiente al comando se vuelve loco, cada tanto se queda quieto.
